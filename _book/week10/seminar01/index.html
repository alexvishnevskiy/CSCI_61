Seminar 1

Dijkstra's algorithm

The algorithm to find the shortest path between two vertices with weighted graph. (Greedy technique)

The algorithm

  1. Distance array/vector

The same length as number of vertices. Initialize everything to infinity except the source, source = 0.

  1. Find the smallest "available" vertex.

"Available" - not infinity, not marked as unavailable. vertex a -> look at all neightbours

  1. Update the distance array/vector using the neighbours.

  2. Mark the vertex as unavailable(visited).

  3. Repeat 2-4 until N-1 iteration, N - number of vertices.
